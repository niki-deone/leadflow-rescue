# LeadFlow 🚀

AI-Powered CRM for modern sales teams. Built with React + Vite.

## Features

- ✅ Full lead management (create, edit, delete, pipeline stages)
- ✅ Real-time sync with Supabase database
- ✅ AI lead scoring powered by GPT-4
- ✅ CSV export for reporting
- ✅ Secure authentication with role-based access
- ✅ Fully responsive - works great on mobile
- ✅ Dark mode UI with glassmorphism design

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- Vite
- Supabase (database + auth)
- OpenAI GPT-4 (lead scoring)
